一、快速入门
    1、获取授权
      默认的版本有使用期限限制，如果想获取授权，请单击“获取注册信息”程序，并将生成的SystemInfo.sys文件发送给软件供应商，
      软件供应商将根据用户提供的硬件信息生成授权文件license.dat，用户收到授权文件后，直接复制到license目录下，覆盖该目录下的文件即可。
      拷贝授权文件后，系统会自动获取授权。
    
    2、安装并启动程序：
      打开安装目录，进入bin目录，点击运行“Install.bat”，进行程序安装。安装完成后，程序在后台运行。
    
    3、登录系统管理界面
	打开浏览器，输入：http://127.0.0.1:8080，默认用户名：admin 密码：123456。为了安全，请admin账户进入后立即修改登录密码。
    
    4、卸载并停止程序
      进入bin目录下，点击执行“Uninstall.bat”，将停止并卸载程序。
    
    5、连接FTP服务器，连接使用的FTP服务器默认的用户名和密码如下：
      用户名:ftpuser
      密码：123456
      端口号：21
    
    6、ftp传输文件目录说明：
      WIFI监控设备或者电子围栏默认传输的ftp目录在程序安装目录下 tmp 目录下，经过网管处理后的文件在logs目录下，如果文件有错误，
      则存放在logs目录下的error目录。当天文件存在在当天日期的目录下，历史文件将打包存放在history目录下。

二、目录结构说明
    
    bin
       系统程序目录，可以进行程序的安装与卸载，以及集成FTP、SFTP、FTPS服务器工具，以及辅助查看测试工具。
    
    conf
       系统配置文件目录。
    
    jdk
       JAVA运行与开发环境，默认是JDK8.0 64位版本。

    license
       授权码。软件的注册与授权文件目录。license查看和注册，包括生成机器码、查看规格以及注册。
    
    tomcat
       tomcat应用服务器。版本为tomcat8.0-x64 64位版本。
    
    mysql
       mysql 数据库和管理工具，基于mysql5.6版本，是64位版本。
	   
    tmp
       临时文件目录，用于接收ftp文件，接收升级文件的临时存放目录。
	   
    logs
       日志文件目录，其中history用于存放每天的日志压缩文件。error存放出现错误的文件。
	   
    update
       “软件更新”功能对应的文件目录（目前没有，仅作保留）
